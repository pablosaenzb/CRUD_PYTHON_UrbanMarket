Luego existe una pagina de estilos generales a todo el sitio que cada pagina carga en su elemento head.

estilos-general.css

Finalmente el header y footer que son comunes a todas las paginas del sitio tambien tienen sus propias paginas de estilo que se cargar también al cargar cada pagina del sitio.

estilos header.css y estilos footer.css


Ya todas las páginas estan interconectadas a traves del menu de navegación dando ya la imagen de un sitio web completo.
